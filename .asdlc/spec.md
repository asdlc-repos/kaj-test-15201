# Overview

A web app where people create and manage their to-do items, organize them into categories, and track due dates.

# Who uses it

- Individual users — manage their personal task lists.

# What they can do

- Create a new task with a title and optional due date.
- Mark tasks as complete or incomplete.
- Organize tasks into categories they create.
- Edit or delete existing tasks.
- See all their tasks in one place, grouped by category.
